# 🔐 Facebook Login Page — UI Clone (Frontend)

Адаптивная копия страницы входа Facebook, реализованная на **HTML и CSS**.  
Проект демонстрирует навыки верстки, работы с Flexbox и современного UI.

---

## 🚀 Демо

👉 https://dias-frontend.github.io/expanding-cards/

---

## 🚀 Возможности

✅ Адаптивная верстка 📱  
✅ Современный UI (похожий на Facebook)  
✅ Форма входа (email / пароль)  
✅ Кнопка Login  
✅ Ссылка "Forgot password?"  
✅ Кнопка "Create new account"  
✅ Hover эффекты ✨  
✅ Чистый CSS без библиотек

## 🎯 Как пользоваться

* 📧 Введи email или номер телефона

* 🔘 Нажми "Log In"  
* 🔗 Используй "Forgot password?"  
* 🆕 Создай новый аккаунт
